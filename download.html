Steps to Get Started

Follow the steps below to include SPIlib in your project:

  • Download the SPIlib binaries from the official website or GitHub repository.
  • Include the source code: Since SPIlib is an open-source library, include the spilib.c directly into your project.
  • Link the library: When compiling your project, make sure to link to libspilib.a using the appropriate compiler flags: 
    gcc game_platformer.c spilib.c -L. -lspilib -lopengl32 -lgdi32 -lglfw3
  • GLFW is already linked inside libspilib.a so you don't need to separately include `glfw3.dll` or `spilib.dll` if they are already inside the static library libspilib.a.
  • Run your game: Now you can build and run your game using the SPIlib environment!
